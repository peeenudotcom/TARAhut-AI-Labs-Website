# Phase 1: Foundation + Tools (Days 1-15)

## Phase 1 Goal
Every student builds a 15+ piece commercial-quality portfolio using AI tools and passes the Phase 1 exam.

---

## DAY 1: What Is AI? The Landscape That Changes Everything

### Learning Objectives
- Define AI, machine learning, and large language models in plain language
- Understand why AI matters for earning money in 2026
- Set up all required accounts
- Write their first AI prompt

### Minute-by-Minute Breakdown

| Time | Block | Activity | Trainer Notes |
|---|---|---|---|
| 0:00-0:05 | Warm-Up | Trainer introduces themselves. Ask: "Kitne logon ne ChatGPT use kiya hai?" (How many have used ChatGPT?). Show of hands. Set the tone: "45 dinon mein aap paise kamana shuru karoge." | Start with energy. Stand, don't sit. Make eye contact with every student. |
| 0:05-0:15 | Context Setting | No recap on Day 1. Instead: student introductions. Each student says name + "Main AI se kya karna chahta/chahti hoon" (what I want to do with AI). Trainer writes goals on whiteboard. | Max 30 seconds per student. If batch >15, do pairs instead. |
| 0:15-0:25 | Instruction | **What is AI?** Explain with analogies: AI = bahut smart assistant jo kabhi thakta nahi (a very smart assistant that never gets tired). Show 3 live demos: (1) ChatGPT answering a question, (2) Canva generating a design, (3) HeyGen creating a video. | Do NOT get technical. No mention of neural networks. Keep it magical. The goal is excitement, not understanding. |
| 0:25-0:35 | Instruction | **The AI Money Map.** Draw on whiteboard: 6 ways people earn with AI today: (1) Content writing, (2) Design, (3) Video creation, (4) Social media management, (5) Marketing for local businesses, (6) Automation. Show 2-3 real examples from India/Punjab. | Use local examples: "Amritsar da ek restaurant, unne AI se menu design karvaya" type stories. Real or realistic. |
| 0:35-0:40 | Instruction | **Course roadmap.** Show the 45-day journey on a visual timeline. Phase 1 = skills, Phase 2 = packaging, Phase 3 = earning. "Day 35 tak aap mein se har koi paise kama chuka hoga." | Print this timeline as a poster. Put it on the wall. Students should see it every day. |
| 0:40-1:10 | Guided Practice | **Account setup marathon.** Every student creates: (1) ChatGPT account, (2) Canva account, (3) Google account if needed. Trainer walks through each step on projector. Students follow on their phones/laptops. Help each student individually. | This WILL take the full 30 minutes. Have TAs help. Some students will struggle with email verification. Be patient. Have backup phone numbers for OTP issues. |
| 1:10-1:15 | Break | 5-minute break. | Remind students to charge devices. |
| 1:15-1:45 | Independent Work | **First prompt challenge.** Students open ChatGPT and complete 5 tasks: (1) "Introduce yourself in English" -- ask AI to improve it, (2) "Write a birthday wish for a friend in Punjabi", (3) "Give me 5 business ideas for someone in [their city]", (4) "Explain AI to a 10-year-old in Hindi", (5) "Write a WhatsApp message to a shop owner offering marketing help." | Circulate. Read prompts over shoulders. Common issue: students will write very short prompts. That's fine for today -- prompt engineering comes Day 4. |
| 1:45-1:50 | Independent Work | **Screenshot and save.** Students screenshot their best ChatGPT output and send to the class WhatsApp group. Trainer picks 3 best and shows on screen. | Celebrate good outputs loudly. First day = build confidence. |
| 1:50-2:00 | Wrap-Up | 3 takeaways: (1) AI is a tool, not magic -- you guide it, (2) There are real ways to earn money with AI starting in weeks, (3) This course will give you the exact skills and steps. **Homework:** Ask ChatGPT to write a 100-word bio about yourself. Save screenshot. **Exit ticket:** "AI meri life kaise change kar sakta hai? Ek line mein likho." | Collect exit tickets physically or via WhatsApp. Read a few aloud. End on a high. |

## DAY 2: How AI Actually Works (LLMs, Tokens, GPUs)

### Learning Objectives
- Understand LLMs, tokens, and context windows at a practical level
- Know the difference between GPT-4, Claude, Gemini, and when to use each
- Understand free vs. paid tiers and what you get

### Minute-by-Minute Breakdown

| Time | Block | Activity | Trainer Notes |
|---|---|---|---|
| 0:00-0:05 | Warm-Up | "Kal raat ChatGPT se kya poochha?" (What did you ask ChatGPT last night?). 3 students share. Celebrate the most creative use. | If nobody used it, no shaming. Just say "Aaj ke baad roz use karoge." |
| 0:05-0:15 | Recap | Quick review: What is AI? What are 6 ways to earn? Check homework: 3 students read their AI-written bios. Class votes on best one. | Give a small prize (candy/sticker) for best bio. Gamification starts Day 1. |
| 0:15-0:25 | Instruction | **How LLMs work -- the restaurant analogy.** LLM = a chef who has read every recipe book in the world. Tokens = ingredients. Context window = the size of the kitchen. More tokens = bigger dish you can cook. Show token counter: type a sentence, show how many tokens it uses. | Use https://platform.openai.com/tokenizer to show tokens live. "Dekhlo, 'Sat Sri Akal' = 5 tokens, 'Hello' = 1 token. Punjabi mein zyada tokens lagte hain." |
| 0:25-0:35 | Instruction | **The AI Tool Landscape.** Show comparison table on screen: ChatGPT vs Claude vs Gemini vs Perplexity. Explain: ChatGPT = best all-rounder, Claude = best for long writing, Gemini = free + Google integration, Perplexity = best for research. Show each tool live for 2 minutes. | Do NOT overwhelm. Message: "ChatGPT sabse pehle seekho, baaki bonus hai." (Learn ChatGPT first, rest is bonus.) |
| 0:35-0:40 | Instruction | **Free vs. Paid: What do you actually get?** Show side-by-side: free ChatGPT vs Plus. Explain: free is enough to start earning. Paid = faster, smarter, more features. "Pehle paisa kamao, phir upgrade karo." (Earn first, then upgrade.) | Do NOT pressure students to buy subscriptions. Many won't be able to afford it. Free tools are sufficient for the course. |
| 0:40-1:10 | Guided Practice | **AI Tool Olympics.** Same question asked to 4 tools: "Write a promotional message for a Punjabi restaurant in Amritsar that just opened. Include a Diwali offer." Students try on ChatGPT (mandatory) + 1 other tool of their choice. Compare outputs. Discuss: which is better? Why? | Pair students: one uses ChatGPT, partner uses Gemini. Compare. This builds critical evaluation skills early. |
| 1:10-1:15 | Break | | |
| 1:15-1:45 | Independent Work | **10-Prompt Challenge.** Students write 10 different prompts across categories: (1) Business email, (2) Social media caption in Hindi, (3) Product description, (4) Customer complaint response, (5) Festival greeting, (6) Job application cover letter, (7) Restaurant menu description, (8) Real estate listing, (9) Wedding invitation text, (10) Thank you message to a teacher. Save all outputs. | Students must complete all 10. This builds prompt muscle memory. Speed matters -- commercial work requires speed. |
| 1:45-1:50 | Independent Work | Students pick their 3 best outputs, polish them, and send to WhatsApp group. | Start building the habit of self-curation. Not everything AI produces is portfolio-worthy. |
| 1:50-2:00 | Wrap-Up | 3 takeaways: (1) AI tools are different -- learn when to use which, (2) Tokens matter -- longer input = better output usually, (3) Free tools are enough to start. **Homework:** Create accounts on Claude and Gemini. Ask each the same question and screenshot the comparison. **Exit ticket:** "ChatGPT aur Gemini mein kya fark hai?" | |

## DAY 4: Prompt Engineering Part 1 -- The 5-Block Framework

### Learning Objectives
- Master the 5-Block Prompt Framework
- Write structured prompts that produce commercial-quality output
- Understand the difference between amateur and professional prompting

### Minute-by-Minute Breakdown

| Time | Block | Activity | Trainer Notes |
|---|---|---|---|
| 0:00-0:05 | Warm-Up | Show two outputs: one from a lazy prompt, one from a structured prompt. Same task, wildly different quality. "Prompt engineering = AI se kaam karwane ki kala." (Prompt engineering = the art of getting work done from AI.) | The contrast should be dramatic. Use a real business example. |
| 0:05-0:15 | Recap | Review homework: show 2-3 student posts on screen. Class critiques: what's good, what could be better. Quick challenge: "Improve this prompt in 30 seconds" -- students rewrite a bad prompt. | |
| 0:15-0:40 | Instruction | **The 5-Block Prompt Framework.** Teach each block with examples: **Block 1: ROLE** -- "You are a professional marketing copywriter for small businesses in Punjab." **Block 2: CONTEXT** -- "The client is a newly opened gym in Jalandhar targeting college students." **Block 3: TASK** -- "Write 5 Instagram captions for their launch week." **Block 4: FORMAT** -- "Each caption: max 150 words, include 1 emoji, end with a CTA, include 5 hashtags." **Block 5: CONSTRAINTS** -- "Tone: energetic and young. Language: Hinglish. Do not mention competitor gyms." Live demo: build a prompt block by block, show how output improves with each block added. | Write the 5 blocks on the whiteboard. Keep them visible ALL session. Students should memorize: ROLE, CONTEXT, TASK, FORMAT, CONSTRAINTS. Acronym: RCTFC. |
| 0:40-1:10 | Guided Practice | **Framework drill.** 5 exercises, one for each block. Trainer gives a scenario, students write the complete 5-block prompt. Scenarios: (1) Restaurant menu descriptions, (2) Salon Instagram bio, (3) Coaching center admission WhatsApp message, (4) Real estate property listing, (5) Wedding planner portfolio intro. Trainer reviews each prompt before student runs it. | Do NOT let students run prompts before trainer approval. The goal is prompt quality, not output quantity. Fix structure first. |
| 1:10-1:15 | Break | | |
| 1:15-1:50 | Independent Work | **Prompt Engineering Portfolio.** Students create a "Prompt Library" document. They write 10 5-block prompts for 10 different local business scenarios. For each prompt: write the prompt, run it, rate the output (1-5 stars), refine if below 4 stars. Final document = 10 prompts with outputs. | This becomes a reusable asset. Students will use this prompt library when they start working with real clients. Emphasize: "Yeh tumhari template library hai. Isse time bachega aur quality badhegi." |
| 1:50-2:00 | Wrap-Up | 3 takeaways: (1) Good prompts = good output, always, (2) The 5-Block Framework works for any AI tool, (3) Build a prompt library -- it's your competitive advantage. **Homework:** Write 5 more prompts for business types not covered in class. **Exit ticket:** Write a complete 5-block prompt from memory for "a dentist clinic launching teeth whitening service." | |

### Portfolio Pieces
- Prompt Library document with 10+ structured prompts and outputs
