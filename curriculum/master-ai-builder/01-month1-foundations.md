## Session 3: CRISP Prompt Engineering — Part 1

**Date:** Week 1, Day 3
**Goal:** Students learn the CRISP framework and immediately write better prompts.

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **AI Fact** — "The difference between a $0 and $50,000 AI consultant? How they write prompts." | Hook: prompting is THE skill. |
| 0:05–0:15 | **Recap** — Review tool comparison. Quick: "Raise hand if Claude won the most rounds for you." Discuss. Share Perplexity homework experiences. | Bridge to today: "Now that you know the tools, let's learn to drive them properly." |
| 0:15–0:40 | **Instruction: The CRISP Framework** — TARAhut's proprietary prompt methodology. **C** = Context (background info, situation), **R** = Role (who should the AI be?), **I** = Instructions (what exactly do you want?), **S** = Specifics (format, length, tone, language, constraints), **P** = Parameters (temperature, style, audience). Demo: Transform a bad prompt into a CRISP prompt live. Show the output difference. | Write CRISP on the board vertically. Each letter gets 3-4 minutes. Use real before/after examples. The transformation should be dramatic. |
| 0:40–1:10 | **Guided Practice: CRISP Drill** — 5 scenarios. For each, students write the prompt using CRISP structure. Trainer does the first one together. Scenarios: (1) Write a WhatsApp message to a client, (2) Create a social media caption for a restaurant, (3) Draft a job application email, (4) Explain blockchain to a 12-year-old, (5) Write a product description for a Punjabi food brand. | Walk room. Check that students are using ALL 5 elements. Common mistake: skipping Context or Parameters. |
| 1:10–1:15 | **Break** | |
| 1:15–1:50 | **Independent Practice** — Complete **Worksheet W01-B: CRISP Prompt Builder**. 10 prompts, each requiring full CRISP structure. Students choose their own topics (relevant to their goals). Save all outputs. Best 3 go into their Prompt Library. | Encourage students to use different AI tools for different prompts. |
| 1:50–2:00 | **Wrap-Up** — "Before-and-after" showcase: 2 students show their worst Session 1 prompt vs their best CRISP prompt today. Class votes on most improved. Preview: "Tomorrow we go advanced — chaining, few-shot, and building your personal Prompt Library." **Homework:** Rewrite 5 of your Session 1 prompts using CRISP. Compare outputs. | |

### Materials Needed

## Session 4: CRISP Part 2 + Prompt Library

**Date:** Week 1, Day 4
**Goal:** Advanced prompting techniques. Students build their personal Prompt Library (Week 1 deliverable).

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **AI Fact** — "OpenAI's top prompt engineers earn $300K/year. This is a real career." | Motivation: prompting is not just a skill, it's a profession. |
| 0:05–0:15 | **Recap** — Review CRISP homework. 2 students show their before/after. Class rates improvement 1-10. | Celebrate improvement, not perfection. |
| 0:15–0:40 | **Instruction: Advanced Techniques** — (1) **Chain-of-thought:** "Think step by step before answering." Demo with a math/logic problem. (2) **Few-shot prompting:** Give 2-3 examples, then ask for more. Demo with writing styles. (3) **Mega-prompts:** Combine multiple instructions into one detailed prompt. Demo: create a complete social media strategy in one prompt. (4) **Prompt chaining:** Output of Prompt 1 becomes input of Prompt 2. Demo: research → outline → draft → polish. | 4 techniques, ~6 min each. Show, don't tell. Every technique gets a live demo with real output. |
| 0:40–1:10 | **Guided Practice: Technique Application** — Students try each technique once. Trainer walks through first example together, then students do their own. Use **Worksheet W01-C: Advanced Prompting Techniques**. | Common issue: students try to use all techniques at once. Coach: "Use the right technique for the task." |
| 1:10–1:15 | **Break** | |
| 1:15–1:50 | **Independent Work: Build Your Prompt Library** — Students create a Google Doc or Notion page titled "My Prompt Library." Organize by category: Professional, Creative, Academic, Personal. Include at least 20 prompts with CRISP structure. This is the **Week 1 Deliverable**. | This is their first portfolio piece. Help with formatting. The library should be something they'll actually use after the course. |
| 1:50–2:00 | **Week 1 Wrap-Up** — Celebrate: "You now know more about AI than 95% of people." Quick wins round: each student names one thing AI did for them this week they couldn't do before. Preview Week 2. **Homework:** Use your Prompt Library in real life this weekend. Add 5 more prompts based on what you actually needed. | Collect: Prompt Library links for assessment. |

**Theme:** Use AI to research, analyze, and make data-driven decisions

## Session 13: Perplexity AI + Research Superpowers

**Date:** Week 4, Day 1
**Goal:** Students master AI-powered research — finding, verifying, and synthesizing information.

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **AI Fact** — "Perplexity AI processes 100+ million queries monthly. It's replacing Google for research." | |
| 0:05–0:15 | **Recap** — LinkedIn posting results from brand kit. Engagement? Comments? | Celebrate public posting. It builds confidence. |
| 0:15–0:40 | **Instruction: AI Research Methods** — (1) **Perplexity AI:** Demo — real-time web search with citations. Compare to ChatGPT (which doesn't browse in free tier). Show: research a market, find statistics, verify claims. (2) **NotebookLM:** Upload documents, get AI-powered summaries and Q&A. Demo with a PDF. (3) **Research workflow:** Question → Perplexity search → Verify sources → Synthesize with Claude → Format output. Demo: Research "AI adoption in Indian small businesses" in 10 minutes. | Key lesson: AI research is FASTER but needs VERIFICATION. Always check sources. |
| 0:40–1:10 | **Guided Practice** — Students research a topic relevant to their local area using Perplexity. Topics: "Digital marketing agencies in Ludhiana," "AI training opportunities in Punjab," "Small business challenges in [their city]." Follow the research workflow. | Walk room. Make sure students are checking citations, not just accepting AI output as truth. |
| 1:10–1:15 | **Break** | |
| 1:15–1:50 | **Independent Work** — Begin the **Week 4 Research Report**. Choose a local business (real one they know). Research: industry trends, competitor analysis, customer demographics, digital presence audit. Use Perplexity + Claude. Write first 2 sections of the report. | This report becomes the foundation for Month 3's client projects. Choose the business carefully. |
| 1:50–2:00 | **Wrap-Up** — Share interesting research findings. Preview: tomorrow is data and spreadsheets with AI. **Homework:** Complete the competitor analysis section of your research report. | |

## Session 14: AI + Google Sheets — Data Without Fear

**Date:** Week 4, Day 2
**Goal:** Students use AI to work with data in spreadsheets — formulas, cleaning, analysis.

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **AI Fact** — "60% of office workers spend 4+ hours/week on spreadsheets. AI cuts that to 30 minutes." | |
| 0:05–0:15 | **Recap** — Research report progress. Quick shares. | |
| 0:15–0:40 | **Instruction: AI-Powered Spreadsheets** — (1) **Ask AI for formulas:** "Write me a Google Sheets formula that calculates profit margin." Never memorize formulas again. (2) **Data cleaning:** Paste messy data, ask AI to clean it. (3) **Analysis prompts:** "Analyze this sales data and tell me the top 3 insights." (4) **Dashboard concepts:** Ask AI to design a simple tracking dashboard. Demo: Take raw sales data from a fictional Punjab shop → clean it → analyze it → create a summary. | Many students fear spreadsheets. Make it approachable. "You don't need to know Excel. You need to know how to ask AI about Excel." |
| 0:40–1:10 | **Guided Practice** — Students open Google Sheets. Trainer provides a sample dataset (10 rows of sales data for a local shop). Together: (1) Use AI to write formulas (SUM, AVERAGE, MAX, IF). (2) Create a simple chart. (3) Ask AI to identify trends. (4) Write a 3-line summary of insights. | Provide the sample data via WhatsApp group or shared Drive link. |
| 1:10–1:15 | **Break** | |
| 1:15–1:50 | **Independent Work** — Create a simple business dashboard for the local business from Session 13's research. Include: revenue tracker (even with dummy data), expense categories, profit calculation, one chart. Use AI for all formulas and formatting. | Not everyone will finish. That's OK. The skill is "I can ask AI to help me with data." |
| 1:50–2:00 | **Wrap-Up** — Show 2 dashboards. Preview: tomorrow is pulling it all together — the complete research + analysis report. **Homework:** Find real data about your chosen business (even approximate — monthly revenue, customer count, social media followers). | |

## Session 15: Research Report Assembly

**Date:** Week 4, Day 3
**Goal:** Students complete their Research Report + Business Analysis.

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **AI Fact** — "Management consultants charge ₹5,000-50,000 for a business analysis report. You're building one for free." | |
| 0:05–0:15 | **Recap** — Real data collection. Who found numbers? Discuss challenges. | "Approximate is fine. We're learning the method, not auditing their books." |
| 0:15–0:40 | **Instruction: Report Structure & AI Assistance** — Show a professional business analysis report template. Sections: (1) Executive Summary, (2) Business Overview, (3) Market & Competitor Analysis, (4) Digital Presence Audit, (5) SWOT Analysis, (6) AI Opportunity Map — where AI can help this business, (7) Recommendations, (8) Data Appendix. Demo: Use Claude to write the Executive Summary and SWOT based on the research collected. | The AI Opportunity Map is unique to this program. It's what makes this more valuable than a generic report. |
| 0:40–1:10 | **Guided Practice** — Students work on their report. Trainer circulates, helps with structure, AI prompts for each section. Focus on Sections 1-4 during guided time. | Common issue: students try to make it perfect. Coach: "Done is better than perfect. Refine later." |
| 1:10–1:15 | **Break** | |
| 1:15–1:50 | **Independent Work** — Complete Sections 5-8. The AI Opportunity Map is the key section: identify 3-5 specific ways this business could use AI tools (from what students learned in Weeks 1-3). Be specific: "Use Canva AI to create weekly Instagram posts instead of paying a designer ₹15,000/month." | This section connects everything. Students prove they can apply AI knowledge to real problems. |
| 1:50–2:00 | **Wrap-Up** — Quick shares: best AI Opportunity from each report. Preview: tomorrow is Month 1 exam + portfolio review. **Homework:** Polish the report. Add formatting, cover page, table of contents. | |

## Session 16: Month 1 Exam + Portfolio Review

**Date:** Week 4, Day 4
**Goal:** Assess Month 1 learning. Review all portfolio pieces. Celebrate progress.

### Minute-by-Minute

| Time | Activity | Trainer Notes |
|------|----------|---------------|
| 0:00–0:05 | **Energy setter** — "One month ago, you didn't know what a prompt was. Today you have a portfolio. Let's prove it." | |
| 0:05–0:35 | **Practical Exam (30 min)** — 5 timed challenges: (1) Write a CRISP prompt for a given scenario (5 min). (2) Create a social media graphic in Canva for a given business (7 min). (3) Research a given topic using Perplexity and write a 100-word summary with sources (6 min). (4) Take a bad email and rewrite it professionally using AI (5 min). (5) Given a business description, identify 3 AI opportunities (7 min). | Timer on screen. Students work individually. No help from trainer or peers. This tests real skills under pressure. |
| 0:35–0:45 | **Exam Review** — Quick review of answers. Show ideal outputs for each challenge. | Not grading on screen. Trainer collects and grades later. |
| 0:45–1:10 | **Portfolio Review** — Each student opens their complete portfolio. Trainer does a quick scan of all pieces: Prompt Library (W1), Writing Portfolio (W2), Brand Kit (W3), Research Report (W4). Checklist review using **Portfolio Scorecard**. | Move fast — 2 min per student. Check: completeness, quality, consistency. Note gaps for individual feedback. |
| 1:10–1:15 | **Break** | |
| 1:15–1:45 | **Month 1 Retrospective** — Group discussion: (1) What was the hardest part? (2) What surprised you most? (3) What AI tool is now your daily driver? (4) What do you want to learn in Month 2? Write 3 "wins" on sticky notes, put on the Wins Wall. | The Wins Wall stays up all 12 weeks. It's a visual progress tracker. |
| 1:45–2:00 | **Month 2 Preview + Celebration** — "Month 2 is where you become a creator. Video, voice, images, automation. The tools that make you truly dangerous." Hand out Month 2 schedule. Group photo. | Share photo in WhatsApp group. Tag: "Month 1 complete. 15+ portfolio pieces. #MasterAIBuilder" |
