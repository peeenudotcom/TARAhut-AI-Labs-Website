Section 02

Bolt.new Deep Dive

Bolt.new is an AI-powered browser tool that writes, runs, and deploys real web applications — all from a single text prompt.

🔗

Open Bolt.new Right Now

Go to bolt.new in your browser. Sign in with Google (free account). You'll see a clean interface — a text box at the bottom. That text box is your entire development environment.

The Bolt.new Interface — What You're Looking At

📄 The Prompt Box (Bottom Center)

This is where you type what you want. Be descriptive. Think of it as giving instructions to a brilliant developer who can build anything — but needs you to tell them what to build.

💻 The Code Editor (Left Panel)

AI generates the code here. You don't need to touch it. But you can see the files being created in real-time — HTML, CSS, JavaScript, all written automatically.

👁 Live Preview (Right Panel)

See your website render instantly on the right side. Every change updates in real-time. What you see is exactly what your visitors will see.

⚙️ The Refine Loop

Type more instructions to refine. "Make the header blue." "Add a contact form." "Change the font to something modern." AI updates the website instantly with each message.

🚀 Deploy in One Click

Click the Share/Deploy button. Bolt.new gives you a live URL ending in .bolt.app that you can share immediately. Your website is now on the internet.

The Art of Describing Your Website

📝

Weak Prompt vs Strong Prompt

Weak: "Make me a website"
Strong: "Build a professional portfolio website for a graphic designer. Include: a hero section with my name and tagline, a portfolio grid showing 6 projects with hover effects, an about section, a skills section with progress bars, and a contact form. Use a dark theme with green accents. Make it mobile-responsive."

The more specific you are, the better the result. Describe the purpose, sections, style, and audience.

Step-by-Step Build Process

01

Start with the Master Prompt

Open bolt.new and paste the complete initial prompt (see below). This creates the entire structure in one go.

Review the First Output

Section 04

Building a Business Landing Page

Real-world exercise: Build a complete landing page for a local Kotkapura business — contact form, map link, services, and everything.

🏪

The Client Brief

Business: "Gill Sweets & Snacks" — a 20-year-old mithai shop in Kotkapura
Problem: Customers can't find them online, WhatsApp orders are messy, no way to show their menu
Budget for Website: ₹0
What They Need: Simple, professional website that looks great on mobile and drives WhatsApp orders

The Build — Prompt by Prompt

Prompt 1: Initial Build

"Build a landing page for Gill Sweets & Snacks, a traditional Indian mithai shop in Kotkapura, Punjab. Include: hero with shop name and tagline 'Authentic Sweets Since 2004', services section showing 6 popular items (Gulab Jamun, Barfi, Ladoo, Halwa, Namkeen, Special Dry Fruit Sweets) with price ranges, about section about the family business, and a contact section. Use warm orange and gold colors on a dark background. Mobile-first design."

Prompt 2: Add Order Button

"Add a prominent 'Order on WhatsApp' button in the hero and in the contact section. The button should open WhatsApp with a pre-filled message: 'Hello! I'd like to place an order from your website.' Use phone number +91 98765-XXXXX. Make the button bright orange with a WhatsApp icon."

Prompt 3: Add Google Maps

"Add a 'Find Us' section with a Google Maps link button that opens maps to 'Gill Sweets, Kotkapura, Punjab'. Include the full address and a landmark. Add business hours: Monday-Saturday 7AM-10PM, Sunday 8AM-9PM."

Prompt 4: Add Contact Form

"Add a simple contact form with fields: Name, Phone Number, Order Details (textarea), Delivery or Pickup (dropdown). Add a submit button. Style the form to match the warm color scheme. Below the form, show the phone number and email as clickable links."

Prompt 5: Polish and Deploy

"Add a floating 'Order Now' button fixed to the bottom of every page on mobile. Add smooth animations on scroll. Add a simple footer with shop name, copyright 2026, and social media icon placeholders for Instagram and Facebook. Make sure all text is clearly readable on both mobile and desktop."

💡

What This Website Would Cost From a Developer

A local web developer in Kotkapura or Faridkot would charge ₹8,000 to ₹20,000 for exactly this website. It would take them 3-7 days to deliver. You just built an equivalent in 30-45 minutes for ₹0.
