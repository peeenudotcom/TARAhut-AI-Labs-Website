Web Page CRISP Prompt Template

CRISP Prompt — Web Page
Context: I need a landing page for [business/course/product name]. Visitors are [audience].
Role: Expert frontend web developer specializing in clean, conversion-optimized landing pages.
Instructions: Build a single HTML file (HTML + CSS + any JS in the same file). Mobile-responsive. No external dependencies except Google Fonts.
Specifics:
- Color scheme: [primary color] and [secondary color]
- Sections: [list each section: hero, features, pricing, CTA, etc.]
- Key content: [headline, subheadline, benefit points]
- Buttons: [WhatsApp number/email/form action]
- Font: Clean, modern (suggest Poppins or Inter from Google Fonts)
Purpose: Generate enquiries / capture leads / showcase the course/product.

Power Iteration Prompts

Use these after your first draft to refine:

  • "Make the hero section taller and add a subtle animated gradient background."
  • "Add a sticky navigation bar with smooth scroll to each section."
  • "Convert the static CTA button to a form that pops up when clicked."
  • "Optimize the mobile layout — the cards should stack on phones."
  • "Add a FAQ section with accordion (click to expand/collapse answers)."

Python Scripting

Simple automation scripts: rename files, process CSVs, send emails. Claude writes it, you run it — no Python knowledge required.

Python: The Automation Superpower Python is the world's most popular automation language. With Claude writing it for you, you can automate repetitive tasks in your business without learning to code. Install Python once → Claude writes the scripts → you run them.

Example 1: Rename Files in Bulk

What to Say to Claude
Write a Python script that:
1. Goes through all files in a folder I specify
2. Renames them following this pattern: YYYY-MM-DD_OriginalName (adds today's date at the start)
3. Only processes .jpg, .png, and .pdf files — skip everything else
4. Shows me a list of what it renamed before actually doing it, and asks "Confirm? (y/n)"
5. Prints a success message when done

Make it beginner-friendly with comments explaining each step. I'll run it on Mac/Windows.

Example 2: Process CSV Data

What to Say to Claude
Write a Python script that:
1. Reads a CSV file called "leads.csv" that has columns: Name, Phone, Email, Course, Date
2. Filters to only rows where Course = "Claude AI"
3. Removes duplicate Phone numbers (keep the first occurrence)
4. Saves the clean data to "leads_claude_clean.csv"
5. Prints: "Processed X leads. Found Y duplicates. Saved Z unique leads."

I have Python installed. Include the pip install command for any libraries needed.

Example 3: Send Emails Automatically

What to Say to Claude
Write a Python script that sends personalized emails to a list of contacts.

It should:
1. Read from "contacts.csv" with columns: Name, Email
2. Send each person an email with: Subject "Your TARAhut AI Labs Course Starts Monday!" and a personalized body that starts with "Dear [Name],"
3. Use Gmail SMTP (I'll add my credentials)
4. Add a 2-second delay between emails to avoid spam flags
5. Log which emails succeeded and which failed to "email_log.txt"

Show me where to put my Gmail address and app password. Include setup instructions.

How to Run Claude's Python Scripts

1

Download from python.org — choose the latest version, check "Add Python to PATH" during install.

2

4

Run the Script

Open Command Prompt → navigate to folder → type: python scriptname.py → press Enter.

5

Debug with Claude

If you get an error, copy the error message → paste to Claude → "I got this error, please fix it." Claude will patch the script.

The Error Loop Getting an error when running Claude's code is NORMAL and expected. Just paste the error back to Claude: "I got this error: [paste error]. Please fix it." Claude will diagnose and fix. Most scripts work within 2-3 iterations.
