Section 1

What is an API?

The simplest explanation: your app talks to Claude's brain directly, without using the website. Here's exactly what that means.

The Phone Call Analogy

Using Claude.ai = Visiting someone's house. You go to their location, ring the doorbell, they answer, you talk, you leave. It's great for casual conversations and exploration.

Using Claude API = Calling them on the phone. You dial their number from anywhere. They answer. You get what you need. The call can be made from your app, your website, your script — anywhere. And you can set it up to call automatically, without you being there.

The API in Plain Language An API (Application Programming Interface) is a way for software to talk to other software. The Claude API is Claude's "phone number" — your code dials it, sends a message, and Claude responds. Your website, app, or script handles the conversation — you just set the rules.

Why Does the API Matter?

🌐

Claude on YOUR Website

Instead of sending customers to Claude.ai, Claude lives inside your website. Your branding, your interface, your control.

🤖

Automatic Without You

API-powered Claude can respond to customer enquiries at 3am while you sleep. No human needed in the loop for routine responses.

🔗

Connected to Your Data

Your app can send Claude context from your database — customer name, purchase history, course enrollment — for truly personalized responses.

⚡

Speed at Scale

One Claude API integration can handle hundreds of conversations simultaneously. No waiting. No queue. Instant AI for every user.

Getting Your API Key

Your API key is your access credential — the unique identifier that lets Claude know it's your app making the request.

What is an API Key? Think of it as your personal password for the Claude phone line. Every time your app calls Claude, it presents this key. Anthropic uses it to: (1) verify you're authorized, (2) track your usage, and (3) apply your billing. Keep it secret — never share it publicly.

Getting Your Key: Step by Step

1

Go to console.anthropic.com

This is Anthropic's developer console — separate from claude.ai. Create an account or sign in with your existing Anthropic account.

2

Navigate to API Keys

In the left sidebar, find "API Keys." Click "Create Key." Give it a descriptive name like "TARAhut Website Chatbot."

3

Copy and Store Securely

The key is shown ONCE. Copy it immediately and store in a secure place (password manager, not a Google Doc). Anthropic cannot show it again.

4

Add a Payment Method

Go to Billing. Add a credit or debit card. You only pay for what you use — no monthly subscription for the API itself.

5

Set Usage Limits

In Settings → Billing, set a monthly spending limit (e.g., ₹2,000/month) so you're never surprised by a large bill.

Pricing Explained

How API Pricing Works

Claude API is priced per "token" — roughly 1 token = 0.75 words. You pay for both what you SEND (input) and what Claude RESPONDS with (output).

Claude 3.5 Haiku (fast, economical): ~$0.80 per million input tokens, ~$4 per million output tokens
Claude Sonnet (balanced): ~$3 per million input, ~$15 per million output

Real example: A customer support chatbot handling 100 conversations/day, each ~300 words total: costs approximately ₹100-300 per month. Very affordable for the value delivered.

Section 3

Your First API Call

The structure of a Claude API request — parameters explained in plain language. Understanding this unlocks everything.

What a Request Looks Like Every API call to Claude has the same structure: you send a JSON message with specific parameters, and Claude sends back a JSON response. Understanding each parameter gives you complete control over Claude's behavior in your application.

A Simple API Request

Claude API Request Structure (Python)
# Install: pip install anthropic
import anthropic

client = anthropic.Anthropic(api_key="your-api-key-here")

message = client.messages.create(
    model="claude-sonnet-4-5",
    max_tokens=1024,
    system="You are a helpful customer support assistant for TARAhut AI Labs. Be warm, professional, and concise.",
    messages=[
        {"role": "user", "content": "What courses do you offer?"}
    ]
)

# Claude's response:
print(message.content[0].text)

Parameters Explained

model
string — required
Which version of Claude to use. "claude-sonnet-4-5" is the recommended balanced model. Use Haiku for speed, Opus for complex tasks.
max_tokens
integer — required
Maximum length of Claude's response. 1 token ≈ 0.75 words. 1024 = ~750 words. Set based on how long responses need to be in your app.
system
string — optional but crucial
The system prompt — Claude's "identity" for this application. Define its role, tone, knowledge, and constraints here. This shapes every response.
messages
array — required
The conversation history. Each message has "role" (user or assistant) and "content" (the text). Include previous messages for multi-turn conversations.
temperature
float 0-1 — optional
Controls creativity. 0 = very consistent, predictable. 1 = more creative, varied. For support bots: 0.3. For creative writing: 0.8-1.0.
stop_sequences
array — optional
Words that tell Claude to stop generating. Useful for structured outputs — stop when Claude writes "END" or "---". Controls output format precisely.

The System Prompt is Your Superpower

The system parameter shapes every single response in your application. A well-written system prompt means Claude stays on-brand, on-topic, and helpful — no matter what users ask.

Powerful System Prompt Example
You are the AI assistant for TARAhut AI Labs, an AI training institute in Kotkapura, Punjab.

Your role: Help website visitors understand our courses, pricing, and enrollment process.
Tone: Warm, knowledgeable, helpful — like a friendly admission counselor.

What you know:
- We offer: Master Claude 15 Days (₹2999), ChatGPT Business (₹1999)
- Next batch starts: [DATE]
- Contact: WhatsApp 9876543210
- We're based in Kotkapura, with online options available

Rules:
- Never make up information not listed above
- For complex questions, say "Let me connect you with our team"
- Always end with a gentle nudge to WhatsApp for enrollment
