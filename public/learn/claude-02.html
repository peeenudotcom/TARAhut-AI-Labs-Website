The Claude Interface Tour

A complete annotated walkthrough of every element in claude.ai. No guessing — know exactly what everything does.

🗺️

Why Interface Mastery Matters

Speed is leverage. A user who knows every shortcut and feature completes the same tasks in half the time. Over a year, interface mastery translates to hundreds of hours saved. This section is about deliberate competence, not accidental discovery.

Left Sidebar — Navigation & History

1

New Chat button — Starts a fresh conversation with no prior context. Use this when switching topics completely. Every new chat starts with Claude's full context window available.

2

Chat History — All previous conversations, searchable. Claude does not use previous chats unless you explicitly paste content into the current chat. Each conversation is isolated.

3

Projects — Persistent workspaces with custom instructions and uploaded documents. The most powerful feature for professional workflows. Covered in depth on Day 4.

4

Starred Conversations — Pin important conversations for quick access. Star your best prompt templates and reference conversations here.

5

Profile & Settings — Account, subscription, default model, and appearance. Check your usage limits here when approaching your plan's threshold.

Main Chat Area — The Core Interface

6

Model Selector — Top of chat. Click to switch between Haiku, Sonnet, and Opus mid-conversation. Model switches apply to the next message only; previous messages stay as-is.

7

Message Input Box — Supports markdown formatting as you type. Press Shift+Enter for new lines. The input box expands automatically with your text. Maximum single message: ~3,000 words.

8

Attach Files — Upload PDFs, documents, images, code files, data files. Claude reads and analyzes these inline. Supported formats: PDF, DOCX, TXT, CSV, most image formats, code files.

9

Send Button / Stop Button — The send button becomes a stop button while Claude is generating. Click stop to halt generation mid-response — useful when Claude goes in the wrong direction.

10

Response Actions — On every Claude response: Copy (copies full response), Thumbs Up/Down (sends feedback to Anthropic), Retry (regenerates with same prompt), Edit (modifies Claude's response directly).

Response-Level Features

📋

Copy Button

Copies the entire response as clean text, stripping markdown formatting. Paste directly into Word, email, or any document. The formatting comes through cleanly.

🔄

Retry / Regenerate

Re-runs the same prompt for a different response. Use this when Claude's first response misses the mark. Different random seeds produce meaningfully different outputs.

✏️

Edit Message

Edit your previous message without starting a new chat. The conversation branches — both the original and edited versions coexist. Useful for testing prompt variations.

🔗

Share

Creates a shareable link to the conversation (read-only). Share with team members, clients, or include in reports. Link recipients can view but not edit the conversation.

Model Switching Mid-Conversation

⚡

Professional Technique: Layered Model Use

Start a conversation with Sonnet to draft and explore ideas quickly. When you have a refined question or need the highest quality for a specific output, switch to Opus for that message only. Then switch back to Sonnet to continue. This hybrid approach gives you quality where it matters without the speed/cost penalty throughout.

📎

File Upload Strategy

Claude accepts files up to 10MB. For best results:
PDFs: Text-based PDFs work best. Scanned PDFs may have OCR limitations.
Images: Claude can read text in images, describe visual content, and analyze charts.
Code files: Paste into the message for better context, or upload for large codebases.
Spreadsheets: Export as CSV for cleanest data analysis. Excel formatting can confuse parsing.
Multiple files: You can upload up to 5 files per message.

Context Window Management

⚠️ Warning Sign

Context Saturation

Long conversations degrade quality. When a conversation has 20+ exchanges, Claude starts losing track of early instructions. Signs: repeating yourself, quality drops, ignoring constraints.

Best Practice

Fresh Start Strategy

For long projects, don't continue one mega-conversation. Create fresh chats for each phase. Paste relevant prior output as context. This is counterintuitive but dramatically improves quality.

Pro Move

Context Compression

Ask Claude: "Summarize the key decisions and context from this conversation in 200 words." Paste that summary into your next fresh chat. Full context, clean slate.

The biggest mistake intermediate Claude users make is never starting a new chat. They keep adding to conversations until Claude is confused and context-saturated. Professional usage means being intentional about when to continue vs. when to start fresh.

Conversation Hygiene Practice

Go through your Claude conversation history. Rename your 5 most useful conversations with descriptive names. Star 2-3 that you reference often. Delete any test or junk conversations. Treat your Claude history like a professional filing system.

Handling Long Outputs

📄

When Claude Truncates (Stops Early)

If Claude stops mid-response, simply type continue or please continue from where you stopped. Claude will pick up exactly where it left off. This works because the entire conversation stays in context. No need to repeat the original prompt.

🔢

Requesting Structured Long-Form Output

For documents over 1,000 words, ask Claude to write in sections:
"Write a 3,000-word report. Complete Section 1 first, then stop and wait for me to say 'continue' before proceeding to Section 2."

This gives you editorial control over each section before Claude writes the next one. The final document is far better than asking for everything at once.

Pro Prompting Techniques

Technique

The Clarifying Ask

Before any complex task, tell Claude: "Before you start, ask me any clarifying questions that would help you do this better." Claude will ask 3-5 excellent questions that improve output quality dramatically.

Technique

Self-Critique

After any important output: "Now critique your response. What are the 3 weakest points? How would you improve them?" Then ask Claude to revise based on its own critique. This consistently improves quality.

Technique

Temperature Control via Language

Claude doesn't have a temperature slider in the UI. Use language instead: "Be more creative and unexpected" for higher variation, or "Be precise and conservative" for more consistent, reliable outputs.

Technique

Format Forcing

Claude defaults to markdown. If you need plain text: "Respond in plain text, no markdown formatting." For tables: "Format this as a clean table." For JSON: "Output as valid JSON only, no explanation."

Technique

The Persona Lock

If you set a role, remind Claude mid-conversation: "Remember, you're responding as a senior McKinsey consultant. Adjust your tone accordingly." Long conversations can cause persona drift — reinforcement helps.

Technique

Iteration via Comparison

Ask Claude to give you 3 versions of something: "Write 3 different versions of this intro paragraph — one formal, one conversational, one punchy." Choose and refine from real options.

⚡

The Single Most Underused Feature

The Edit Message feature. Instead of retyping improved prompts as new messages, click edit on your original message and improve it. This keeps conversation history clean and gives you a direct A/B comparison of prompt versions. Professional Claude users edit prompts 3-4 times before accepting a response.

A power user doesn't just use Claude — they direct Claude. Every technique here is about increasing your control over the output.

Power User Practice

Try this sequence on any work task today:
1. Start with the clarifying ask — let Claude identify what it needs to know
2. Answer its questions, then let it produce the output
3. Ask for self-critique
4. Request a revision based on the critique
Compare the final result to what you'd have gotten without this process.

Platform-Specific Best Practices

🌐

Web App (Primary Workflow)

Your main workstation for Claude. Use for:

  • Document analysis and research
  • Building and managing Projects
  • Creating and iterating Artifacts
  • Complex, multi-turn conversations
  • Long-form writing and editing
📱

Mobile App (On-The-Go)

Best for quick, contextual tasks:

  • Quick question/answer during meetings
  • Voice-to-text prompting (use your phone's keyboard)
  • Reading and reviewing existing chats
  • Quick email or message drafts
  • Checking previous conversations on the go
🖥️

Desktop App (Power Users)

The professional's choice for integrated workflows:

  • Global hotkey summons Claude from anywhere
  • Runs alongside other apps seamlessly
  • No browser tab management needed
  • Better for context switching between tools
  • Available for Mac; Windows version in development
💡

The Desktop App Advantage

The Claude desktop app's global hotkey means you can be in any application — Excel, Word, your browser, your email — and summon Claude instantly without switching tabs. This dramatically reduces friction for quick tasks. Set this up on Day 2 and never look back.
