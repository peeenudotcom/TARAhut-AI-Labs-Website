The Best Analogy: Building a House

Think of a website like a house:
🧱 HTML = The Skeleton/Structure — walls, rooms, doors. It defines WHAT is on the page (headings, paragraphs, images, buttons).
🎨 CSS = The Paint & Decoration — colors, fonts, spacing. It makes it LOOK good.
⚡ JavaScript = The Electricity — makes things move, click, and respond. Adds BEHAVIOR.
For your first web page, you mainly need HTML + some CSS. AI does the rest! 🚀

👀 What HTML Actually Looks Like

📝 A tiny piece of HTML — this creates a heading and a paragraph
<h1>My Name is Priya!</h1>
<p>I am 15 years old and I love cricket and coding.</p>
<p>My dream is to become an AI engineer at Google.</p>
✨

The Magic Rule of HTML

Everything in HTML uses tags. A tag opens with <tagname> and closes with </tagname>. Common tags:
<h1> = Big heading  |  <p> = Paragraph
<img> = Image  |  <a> = Link
<button> = Clickable button  |  <div> = A container box
You just learned half of what you need to read HTML! 🎉

The Plan: Your Personal Profile Page

We're going to build a personal profile page with: your name and photo, a short "About Me," your hobbies, your favourite subjects, and links to your social media. This is like having your own mini-website! And it's yours to show colleges, competitions, and relatives. 😎

📋 Step-by-Step: Use These Prompts!

1

Start: Generate the Basic Page

Open ChatGPT. Copy this prompt (fill in YOUR details):

"Build a complete personal profile HTML page for me. My name is [YOUR NAME], I'm [AGE] years old, studying in Class [X] at [SCHOOL NAME]. My hobbies are [HOBBIES]. My favourite subject is [SUBJECT]. My favourite movie/show is [MOVIE]. My dream career is [CAREER]. Use a [DARK BLUE / PURPLE / GREEN] color theme. Make it look modern, clean, and mobile-friendly. Include a fake profile picture placeholder."
2

Copy the Code

ChatGPT will give you a block of HTML code. Click "Copy" in the top right of the code block.

3

See It Live — Instantly!

Go to codepen.io (free, no signup needed). Paste the code in the "HTML" box on the left. Your page appears on the right — INSTANTLY! 🎉

Alternative: Use notepad.exe → paste code → save as "mypage.html" → open in browser.

4

Share it Online (Optional Bonus!)

Sign up at netlify.com or github.io (free). Upload your HTML file. Get a real web link like yourname.netlify.app. Share it everywhere! 🔗

💡

What If Something Looks Wrong?

"The heading color is too dark, change it to bright cyan."
"Add a section for my favourite quotes."
"Make the page look more like Instagram's profile page."
AI is your developer — just give clear instructions! 🎯
