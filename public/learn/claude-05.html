Section 01

What Are Artifacts?

Artifacts are Claude's ability to create standalone, interactive content that appears in a dedicated preview panel. Think of them as Claude's creative workspace.

💡

Artifacts Explained

When Claude creates code, a document, or a visualisation, it can display it as an Artifact — a separate, interactive panel alongside the chat. Unlike regular text responses, Artifacts are live, interactive, and editable. You can preview HTML pages in real-time, copy code with one click, download documents, and share Artifacts via a link. Artifacts turn Claude from a text generator into a full-stack creative tool.

Type🌐

HTML/CSS/JS Pages

Claude creates complete, interactive web pages with styling and functionality. Business cards, landing pages, calculators, forms, interactive charts — all rendered live in the preview panel. You see exactly what the page looks like before downloading.

Type📄

Documents & Text

Long-form documents like reports, proposals, contracts, and articles appear as formatted Artifacts. Better than regular text because they have proper formatting, are easy to copy entirely, and can be downloaded as files.

Type📊

Data Visualisations

Claude creates charts, graphs, and dashboards using SVG or JavaScript. Bar charts, pie charts, line graphs, interactive dashboards — all rendered live. Upload your data and get a visual representation instantly.

Type💻

Code Files

Full code files in Python, JavaScript, HTML, CSS, SQL, and more. Displayed with syntax highlighting, line numbers, and one-click copy. Perfect for developers who need working code snippets or complete scripts.

Type⚙

React Components

Claude creates interactive React components with state management, event handling, and styled UI. Functional widgets, forms, dashboards — all rendered live. The most advanced Artifact type.

You now know the 5 types of Artifacts Claude can create. Each one turns a text conversation into something tangible.
🌟

When to Use Artifacts vs Regular Text

Use Artifacts when: you need something visual (web page, chart), something interactive (calculator, form), something downloadable (document, code file), or something shareable (landing page, report).
Use regular text when: you need a quick answer, short explanation, brainstorming, or conversational back-and-forth. Not everything needs an Artifact.

Enable Artifacts

Artifacts should be enabled by default on Claude Pro. If you don't see the Artifact panel, go to Settings → Feature Preview and ensure Artifacts is turned on. On the free tier, Artifacts may have limited availability.

Iteration & Refinement

The first version of an Artifact is never the final version. The real magic happens in iteration — refining through conversation.

🔄

The 4-Round Artifact Refinement Process

Round 1: Generate the initial Artifact with a detailed CRISP prompt.
Round 2: Review and fix functionality — "The button doesn't work" or "Add a search feature."
Round 3: Polish the design — "Make the font larger," "Change colours to match my brand," "Add more spacing."
Round 4: Final touches — "Add a footer with my contact info," "Make it mobile-responsive," "Add a loading animation."
