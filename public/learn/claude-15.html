The Brief: Kotkapura Clinic Modernization

Dr. Harpreet Singh runs a busy family practice clinic in Kotkapura, Punjab. He sees 40-60 patients daily. His biggest problems: patients don't know what to expect, appointment scheduling is chaotic via phone calls, and he can't communicate in Hindi or Punjabi at scale. He wants to modernize — without a big IT budget.

DELIVERABLE 1

Patient FAQ Claude Project

Build a Claude Project for the clinic with: system prompt defining Claude as "Clinic Assistant," uploaded FAQ covering: appointment timings (Mon-Sat 9am-2pm, 5pm-8pm), fees (consultation ₹300, follow-up ₹150), common specialties, emergency guidance. The Project should respond warmly in Hindi or English based on the patient's language.

DELIVERABLE 2

Appointment Confirmation Artifact Templates

Create Claude Artifacts for 3 appointment scenarios: (A) New Patient Confirmation, (B) Follow-up Reminder, (C) Appointment Rescheduling Notice. Each in English + Hindi. Formatted for WhatsApp sending. Include clinic name, doctor's name, time, what to bring, directions.

DELIVERABLE 3

Hindi & Punjabi Communication Templates

Using CRISP prompts, generate: (A) Hindi post-visit care instructions for common conditions (fever, cold, injury), (B) Punjabi WhatsApp broadcast for seasonal health tips, (C) Hindi appointment reminder script for the clinic's WhatsApp Business account. All culturally adapted for Punjabi patients.

DELIVERABLE 4

API Integration Plan

Write a 1-page document (using Claude) explaining: How the clinic could use Claude API to power a WhatsApp chatbot for appointment booking. Include: what the system prompt would say, estimated cost for 200 patient interactions/month, what Zapier automation would look like, and what the clinic needs to get started (API key, budget, developer).

Why This Brief? This is a real problem. Dr. Harpreet's clinic might be fictional, but clinics, schools, shops, and small businesses with exactly these challenges exist on every street in Punjab. Completing this challenge means you can solve real problems for real businesses — starting this week.
Time Allocation This project is designed for 90 minutes. Each deliverable should take approximately 20-25 minutes. If you get stuck, ask Claude: "I'm working on [deliverable] for a clinic project. Help me structure this."

Executing Deliverable 1: The Claude Project

Clinic Project System Prompt
## Clinic Assistant — Dr. Harpreet's Family Practice, Kotkapura

You are the AI assistant for Dr. Harpreet Singh's family practice clinic in Kotkapura, Punjab.

Your role: Help patients with information about the clinic — timings, fees, appointments, directions, and general health process guidance. You do NOT provide medical advice or diagnosis.

Clinic Information:
- Timings: Mon-Sat: 9am-2pm and 5pm-8pm. Sundays: 10am-12pm (emergency only)
- Fees: New consultation ₹300, Follow-up ₹150, Emergency after-hours ₹500
- Location: [Clinic address, Kotkapura]
- WhatsApp for appointments: [Number]

Language: Respond in the same language the patient uses. Hindi → reply in Hindi. Punjabi → reply in Punjabi. English → reply in English. Use आप (aap) and ਤੁਸੀਂ respectfully.

Important: Always end with "For medical questions, please visit the clinic or call [number]." You cannot diagnose or prescribe.

Executing Deliverable 2: Appointment Templates

Appointment Template Prompt
Create a Claude Artifact with 3 WhatsApp message templates for Dr. Harpreet's clinic:

Template A — New Patient Confirmation (English + Hindi):
Include: patient name placeholder, appointment date/time, what to bring (Aadhaar, previous reports if any), clinic address, "Reply 1 to confirm, 2 to reschedule."

Template B — Follow-up Reminder (English + Hindi):
Include: previous visit date, follow-up appointment, medication reminder, "Take care and rest well."

Template C — Rescheduling Notice (English + Hindi):
Include: apology for inconvenience, new suggested slots (3 options), how to confirm. Warm, professional, not corporate.

Format each as a WhatsApp message: Under 200 words, use emojis appropriately (1-2 max), paragraphs for readability on mobile.

The Approach (60 seconds)

Explain your strategy: "I decided to use 4 Claude features that work together: a Project for consistent clinic identity, Artifacts for ready-to-use templates, CRISP prompts for multilingual content, and an API integration plan for future automation." One sentence per deliverable.

Claude Features Used (60 seconds)

Name and briefly explain each feature: "Claude Projects gave the clinic AI a consistent identity. Artifacts created reusable WhatsApp templates. Live Demo (90 seconds)

Show, don't tell. Open your Claude Project and ask it a patient question — in Hindi. Show the appointment template in Devanagari. Read one line of the Punjabi health tip. Let the audience see it working. Real-time demonstration beats any explanation.
